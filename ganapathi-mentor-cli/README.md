# Ganapathi Mentor AI – Terminal Edition

A legendary senior engineer, architect, and AI mentor living inside the command line.

## Installation

```bash
npm install -g ganapathi-mentor-cli
```

## Usage

```bash
ganapathi-mentor "Explain the architecture of this project"
```

### Modes
- `ganapathi-mentor mentor "<prompt>"`: Architecture and design questions.
- `ganapathi-mentor fix "<prompt>"`: Locate and fix bugs.
- `ganapathi-mentor plan "<prompt>"`: Turn ideas into implementation plans.
- `ganapathi-mentor ops "<prompt>"`: Help with DevOps and infrastructure.

## Requirements
- Node.js 18+
- Mistral API Key
